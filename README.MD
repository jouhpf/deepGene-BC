# deepGene-BC
a deep learning-based breast cancer subtype prediction via somatic point mutation profiles
## Requirements
* `scipy~=1.15.0`
* `numpy~=2.2.4`
* `pandas~=2.2.3`
* `joblib~=1.4.2` 
* `tqdm~=4.67.1`
* `scikit-learn~=1.6.1`
* `seaborn~=0.13.2`
* `matplotlib~=3.10.0`
* `torch~=2.7.0`

all requirements were listed in `requirements.txt`
# Usage
**Entrance**: Run `notebooks/inference.ipynb` for an example use of the model.This script can automatically select a GPU or CPU for inference.
